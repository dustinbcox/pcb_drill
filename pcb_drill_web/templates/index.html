Welcome to pcb_drill_web interface on the Raspberry Pi

This web application is designed to produce gcode to drill a pcb based on a solder mask and a captured image of the pcb drill bed.

The overall process is: one, two, three, easy-z:

  1. Upload and process solder mask to find the solder to drill
  2. Adjust the drill matrix based on a live captured image
  3. Produce gcode to drill the board
